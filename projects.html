Smart Grid Failure Detection using ML

This project predicts smart grid stability using machine learning. Exploratory Data Analysis was performed on generator reaction time, power consumption and price elasticity features. Models such as Random Forest and XGBoost were trained and evaluated using classification metrics and confusion matrix.

The XGBoost model achieved approximately 92–94% accuracy, showing the effectiveness of machine learning for power grid instability detection.

Predictive Maintenance in Military Robotic Systems

A deep learning based predictive maintenance system designed to detect potential failures in military robotic systems using sensor data such as temperature, vibration and pressure.

A GRU (Gated Recurrent Unit) deep learning model was implemented to capture temporal patterns in sensor data. The model achieved about 80.3% prediction accuracy and successfully identified failure cases.
