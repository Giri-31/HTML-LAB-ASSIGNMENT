Artificial Intelligence & Machine Learning

Smart Grid Failure Detection using Machine Learning

This project predicts smart grid stability using machine learning techniques. The dataset includes generator reaction time, power consumption and price elasticity features representing a simulated electrical grid.

The workflow included Exploratory Data Analysis, model training using Random Forest and XGBoost, and evaluation using classification metrics.

The XGBoost model achieved around 92–94% accuracy, demonstrating the effectiveness of machine learning for detecting grid instability.

Predictive Maintenance in Military Robotic Systems

This project focuses on detecting potential failures in military robotic systems using sensor data such as temperature, vibration and pressure.

A deep learning model based on Gated Recurrent Units (GRU) was implemented to capture temporal patterns in time-series data and predict failures before they occur.

The GRU model achieved approximately 80.3% prediction accuracy and successfully identified system failure cases.

Java & Database Systems

Goods Vehicle Management and Booking System

A Java-based logistics management system designed to streamline goods transportation and vehicle booking operations.

The system supports three user roles: Admin, Driver and Customer. Admins manage trips and drivers, customers request transportation, and drivers update trip status.

Built using Java Swing for the interface and MySQL for database storage, the system demonstrates database integration, trip scheduling and logistics management.

Physics / Computational Mechanics

Thermal Stress Evaluation of Heat Sink Fins

This project evaluates the thermal and structural performance of heat sink fins used in CPU cooling systems using simulation.

Different fin geometries such as rectangular fins, circular pin fins and folded fins were modeled and simulated using ANSYS Workbench.

The analysis compared aluminium and copper materials to study temperature distribution, thermal stress and structural deformation in heat sink designs.

Electronics & IoT Systems

IoT-Based Digital Thermometer

An IoT-enabled digital thermometer designed for real-time temperature monitoring and wireless data transmission.

The system uses an LM35 temperature sensor, Arduino Uno for data processing, and an ESP32 Wi-Fi module to send temperature readings to a cloud dashboard for remote monitoring.

Water Level Sensing and Automated Replenishment Controller

This project implements an automated water level monitoring system to control a water pump for automatic tank filling.

The system uses an IC555 timer, BC547 transistors, a relay and LED indicators to detect tank water levels and automatically switch the motor ON or OFF.

It prevents overflow and dry running of pumps while reducing manual monitoring in household water tanks.
