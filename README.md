This portfolio website is organized into multiple HTML files and a shared CSS file for styling

File structure is as follows:

a.Index.html

The main landing page of the portfolio which contains Introduction(Hero section), About me, Skills, Languages, Workshops, Contact information.

b.Projects.html

Displays all the projects which are categorized into AI&ML, Electronics, Java&SQL, Mechanics, Ongoing projects. 

c.Intership.html

A dedicated page for internship opportunities which includes internship details, domains and responsibilities, eligibiltiy criteria and application form

d.style.css

Contains all styling for the website including Layout design, Responsive design, Card components, Navigation and sections

e.Projects folder

Folder containing project images and reports.

Navigation Structure

The header navigation bar allows users to move between pages:

Home --> index.html
Projects --> projects.html
Internship --> internship.html
Contact --> scroll within homepage

Responsiveness

The website is responsive and adapts to different screen size using:

CSS Flexbox, CSS Grid, Media queries

Summary

The website follows a modular structure where each page focuses on a specific purpose, making it easy to navigate, maintain, and scale.
